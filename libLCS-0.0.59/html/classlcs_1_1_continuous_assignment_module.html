Detailed Description

template<int outbits, typename InExprType, unsigned int delay>
class lcs::ContinuousAssignmentModule< outbits, InExprType, delay >

A class (a lcs::Module derivative) whose objects are created as a result of a continuous assignments. A continous assignment through the function lcs::Bus::cass is translated into a module which is of type lcs::ContinuousAssignmentModule. The busses involved in the continuous assignment expression are treated as input busses of this module. The bus to which the continuous assignment is made is treated as the output bus of the module. This module is hidden from libLCS users. The users cannot/will not know about such a creation, and hence will never need to use this class explicitly.
