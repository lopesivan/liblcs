Detailed Description

Every circuit element (like an AND gate) which needs to be driven by lcs::InputBus objects has to be derived from this class Module. User of libLCS desirous of defining his/her own element should subclass this class and redefine the virtual function Module::propogate.

virtual void lcs::Module::onNegEdge ( int  portId  )  [inline, virtual]

A subclass of the class Module should override this function so that it can be notified of a negetive edge on its input busses. If a lcs::LINE_NEG_EDGE event occurs on an input bus, the bus will call this function on the corresponding module object as a notification of the occurance of the event.

Reimplemented in lcs::Counter< bits, event >, lcs::DFlipFlop< type, delay >, lcs::FrequencyDivider< factor, delay >, lcs::JKFlipFlop< type, delay >, lcs::Register< bits, edgeType, delay >, and lcs::ShiftReg< bits, edgeType, shift, delay >.
