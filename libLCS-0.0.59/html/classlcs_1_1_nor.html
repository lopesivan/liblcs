Detailed Description

template<unsigned int width = 1, unsigned int delay = 0>
class lcs::Nor< width, delay >

A template class which encapsulates a logical NOR gate. The first template parameter width denotes the number of lines in the input data bus. In other words, the template parameter width denotes the number of inputs to the NOR gate. Hence, one can instantiate an N-input NOR gate by setting the template parameter value to N. The second template parameter indicates the propogation delay from the input to the output of the NOR gate.

Member Function Documentation

template<unsigned int width, unsigned int delay>
void lcs::Nor< width, delay >::onStateChange ( int  portId  )  [virtual]

Propogates the input data line states to the output. This function is called by the input bus lines to notify the gate of a lcs::LINE_STATE_CHANGE event. When notified, a NOR operation is performed on the line states of the data lines of the input bus and the result is propogated to the single line output bus.

Reimplemented from lcs::Module.
